# Aparna Telecom E-commerce Project

A modern e-commerce platform for Aparna Telecom, featuring a responsive frontend and MongoDB backend system.

## 🚀 Features

- Responsive web design
- Product catalog with categories
- Interactive user interface
- MongoDB database integration
- Modern styling and animations

## 🛠️ Tech Stack

- Frontend:
  - HTML5
  - CSS3
  - JavaScript
- Backend:
  - Node.js
  - MongoDB
  - Express.js

## 📦 Prerequisites

- Node.js installed on your system
- MongoDB installed and running locally
- Git for version control

## 🚀 Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/mrityunjay-pandey/dbms_project.git
cd dbms_project
```

2. Install dependencies:
```bash
npm install
```

3. Start MongoDB:
```bash
# Make sure MongoDB is running on your local machine
# Default port: 27017
```

4. Start the backend server:
```bash
cd ecommerce-backend
node server.js
```

5. Open the frontend:
- Open `index.html` in your web browser to view the main page
- Open `index2.html` in your web browser to view the secondary page
